PDF Token Counter, File Reducer & Splitter

What does this app do?

  • Allows upload of any size PDF file
  • Removes all images and converts each page to text only
  • Reduces file size up to 90%
  • Counts tokens used
  • Splits PDF every 10,000 tokens (This can be modified by user)
  • Shows final token count
  • Generates .zip file of split pdf(s)
  • Creates download link for .zip file

How do I use this app?

    Optional: Change 10000 to desired PDF split.
  1. Click Choose File button below, browse and select desired PDF.
  2. Click Process PDF button and wait while until process completes
  3. Once completed, view Total token count
  4. Finally, click Download PDF Zip Split File zip file!
